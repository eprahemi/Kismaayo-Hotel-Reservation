Setup Progress 0 / 8 steps done
Windows Fedora Linux
Download Java for Windows
Go to adoptium.net and download the Java 17 or 21 MSI installer for Windows (x64). You can also use Oracle Java.
🌐 adoptium.net → Download Java 17 MSI 📋 Copy
Install Java
Double-click the downloaded .msi file. Click Next → Next → Install. Leave all default settings.
⏳ Takes about 1 minute. No special settings needed.
Open Command Prompt
Press Windows Key + R, type cmd, press Enter. A black terminal window opens.
⌨️ Windows Key + R → type "cmd" → Enter 📋 Copy
Verify Java is Installed
In the Command Prompt, type:
C:\Users\You> java -version 📋 Copy
You should see something like: java version "17.0.12" 2024-07-16 LTS
✅ If you see a version number — Java is ready!
Navigate to the Project Folder
Find where you saved the HotelReservation folder, then type:
C:\Users\You> cd Desktop\HotelReservation\java 📋 Copy
(Change the path if your folder is in Downloads or Documents)
Compile the Server
Type this command:
C:\...\java> javac HotelServer.java 📋 Copy
✅ No error messages = it compiled successfully! You'll just see a new prompt.
💡 If you see 'javac' is not recognized, you installed JRE instead of JDK. Download the JDK version.
Start the Server
Type this command:
C:\...\java> java HotelServer 📋 Copy
You should see:
========================================
  Kismaayo Hotel Server running on
  http://localhost:8080
========================================
⚠️ Keep this window open! The server stops if you close it.
Open the Website 🎉
Open your browser and type this in the address bar:
🌐 http://localhost:8080 📋 Copy
🎉 Kismaayo Hotel home page should appear!
Login with: admin / admin123
Install Java (if not installed)
Open a terminal and run:
$ sudo dnf install java-latest-openjdk 📋 Copy
✅ After install, verify with java -version
➡️
Navigate & Compile
$ cd HotelReservation/java && javac HotelServer.java 📋 Copy
🚀
Run the Server
$ java HotelServer 📋 Copy
========================================
  Kismaayo Hotel Server running on
  http://localhost:8080
========================================
🎉 Open browser → http://localhost:8080
Login: admin / admin123

🐛 Troubleshooting

❌
"javac is not recognized" (Windows)
You installed JRE (Runtime), not JDK (Development Kit). Download the JDK version from adoptium.net.
❌
"java is not recognized" (Windows)
Java is not installed. Download and install Java, then restart your computer.
⚠️
Port 8080 already in use
Windows: Open Task Manager → find the program using port 8080 → close it.
Fedora: Run fuser -k 8080/tcp in terminal.
❌
Browser shows "Server error!"
You opened the HTML file directly (file://), not through the server. Make sure the server is running and type http://localhost:8080.
ℹ️
Data files corrupted
Delete the file in data/ folder and restart the server — it recreates default files automatically.
🔒
Can't log in
Default credentials: admin / admin123. If you changed the password, check data/users.txt.
