❌ The Problem
• Hotels in Kismaayo relied on paper records.
• No online booking system was available.
• Customers had to call or visit in person.
• Hard to manage reservations & rooms.
• No way to check availability remotely.
✅ Our Solution
• Full online hotel reservation system.
• Built with pure Java HTTP server.
• Simple .txt file storage — no database.
• Easy booking with live pricing calculator.
• Admin panel to manage everything.
